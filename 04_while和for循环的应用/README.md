# 二、while循环

## 1. 基础语法

``` python
while 条件:
    循环执行代码
    修改循环变量
```

while 会在条件为 True 时持续执行。

示例：

``` python
i = 0

while i < 100:
    print("执行循环")
    i += 1
```

------------------------------------------------------------------------

## 2. 避免死循环

while 不会自动改变循环变量，需要手动更新。

错误：

``` python
i = 1

while i <= 10:
    print(i)
```

由于 i 永远不会变化，会造成无限循环。

正确：

``` python
i = 1

while i <= 10:
    print(i)
    i += 1
```

------------------------------------------------------------------------

# 三、while循环实际应用

## 1. 累加器

通过循环完成累计计算。

例如：

计算1到100的和：

``` python
i = 1
sum_number = 0

while i <= 100:
    sum_number += i
    i += 1
```

核心思想：

-   i：循环变量
-   sum_number：累计结果

------------------------------------------------------------------------

## 2. 布尔变量控制循环

通过 True / False 控制循环状态。

例如猜数字：

``` python
flag = True

while flag:
    if 猜对:
        flag = False
```

当条件满足时改变状态，结束循环。

------------------------------------------------------------------------

## 3. 循环次数统计

``` python
count = 0

while 条件:
    count += 1
```

可以用于：

-   统计执行次数
-   用户操作次数
-   数据处理数量

------------------------------------------------------------------------

# 四、嵌套循环

循环里面可以继续使用循环。

结构：

``` python
while 条件:
    while 条件:
        执行代码
```

注意：

内层循环变量需要重新初始化。

------------------------------------------------------------------------

# 五、九九乘法表

通过嵌套循环实现：

``` python
i = 1

while i <= 9:

    j = 1

    while j <= i:
        print(f"{j} * {i} = {j*i}")
        j += 1

    i += 1
```

核心：

-   外层循环控制行
-   内层循环控制每行内容

------------------------------------------------------------------------

# 六、for循环

## 1. 基础语法

``` python
for 临时变量 in 数据:
    执行代码
```

for主要用于遍历数据。

例如：

``` python
names = ["张三","李四"]

for name in names:
    print(name)
```

------------------------------------------------------------------------

# 七、range函数

range用于生成数字序列。

## 1. range(num)

``` python
range(5)
```

生成：

    0 1 2 3 4

不包含结束值。

------------------------------------------------------------------------

## 2. range(start,end)

``` python
range(1,5)
```

生成：

    1 2 3 4

------------------------------------------------------------------------

## 3. range(start,end,step)

``` python
range(1,10,2)
```

生成：

    1 3 5 7 9

step表示步长。

------------------------------------------------------------------------

# 八、for循环嵌套

for循环也可以进行嵌套：

``` python
for i in range(3):

    for j in range(3):
        print(i,j)
```

应用场景：

-   多维数据处理
-   表格数据遍历
-   图形打印

------------------------------------------------------------------------

# 九、break和continue

## break

作用：

立即结束当前循环。

示例：

``` python
while True:

    if 条件:
        break
```

------------------------------------------------------------------------

## continue

作用：

跳过本次循环，进入下一次。

示例：

``` python
for i in range(10):

    if i == 5:
        continue

    print(i)
```
