## 核心知识点

### 1. Python 文件

Python 源代码文件通常使用 `.py` 作为扩展名，例如：

```text
hello.py
```

### 2. `print()` 函数

`print()` 用来把内容输出到终端：

```python
print("Hello, World!")
print("我正在学习 Python")
print(100)
```

字符串需要使用单引号或双引号包裹，数字可以直接输出。

### 3. 程序执行顺序

Python 默认从上到下逐行执行代码：

```python
print("第一行")
print("第二行")
print("第三行")
```

## 易错点

- 文件扩展名应为 `.py`，不要保存成 `.py.txt`
- 英文括号和引号必须成对出现
- `print` 必须小写，Python 区分大小写
- 中文符号不能代替英文代码符号
